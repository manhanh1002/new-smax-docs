# Báo cáo Tổng thể Module & Tính năng Smax AI

*Ngày tạo: 7/4/2026*

Ok, mình đã rà soát toàn bộ danh sách bạn cung cấp và xác định các **MODULE TÍNH NĂNG chính** của Smax AI, sau đó phân loại vào 5 nhóm theo yêu cầu.  
Mình cũng phân tích 3 khía cạnh **Newbie SME / Pro Mindset / Enterprise** cho từng module.

Dưới đây là bảng tổng hợp:

| Module / Tính năng chính | Nhóm phân loại | Newbie SME | Pro Mindset | Enterprise |
|---|---|---|---|---|---|
| **Kết nối Facebook, Instagram, Zalo OA, Zalo User, Telegram, WhatsApp, Shopee, TikTok User, Website, Phone Channel** | Kết nối kênh | Dễ kết nối qua hướng dẫn, không cần code; chỉ cần quyền admin kênh. | Hiểu rõ giới hạn API từng nền tảng, tối ưu cấu hình để tránh mất kết nối; biết cách mapping dữ liệu khách hàng. | Tư vấn giải pháp đa kênh, đồng bộ CRM, thiết lập hạ tầng omnichannel cho doanh nghiệp lớn. |
| **Module Nhắn tin (Live Chat)** | Tính năng chính | Giao diện trực quan, dễ trả lời khách; không cần kỹ thuật. | Tư duy phân loại hội thoại, gắn tag, phân quyền supporter; kết hợp automation. | Tích hợp vào hệ thống CSKH tổng thể, kết hợp chatbot & livechat, SLA tracking. |
| **Module Bot-Auto (Chatbot, Trigger, Broadcast, Kịch bản chăm sóc)** | Automation | Có thể dùng template sẵn, kéo thả block; cần làm quen logic bot. | Thiết kế kịch bản phức tạp, dùng bộ lọc nâng cao, sequence, API call; tối ưu tỉ lệ chuyển đổi. | Xây dựng hệ thống chăm sóc khách hàng tự động đa kênh, cá nhân hóa theo hành vi. |
| **Module Khách hàng (CRM mini)** | Tính năng chính | Dễ tìm kiếm, lọc khách; import/export file Excel. | Tư duy phân khúc (segment), gắn tag động, đồng bộ dữ liệu từ nhiều nguồn. | Tích hợp CRM lớn, phân tích hành vi, đồng bộ dữ liệu marketing & sales. |
| **Module Thống kê (Dashboard)** | Tính năng chính | Xem báo cáo cơ bản; không cần kỹ thuật. | Phân tích số liệu để tối ưu chiến dịch, đo KPI từng kênh/nhân sự. | Kết nối BI tools, tạo báo cáo tùy biến cho quản trị cấp cao. |
| **Module Cài đặt (Doanh nghiệp, Kênh, Thành viên, API, Tên miền)** | Tính năng chính / Integration | Cần làm theo hướng dẫn; một số mục (API, domain) cần kỹ thuật. | Hiểu cấu trúc hệ thống, phân quyền, bảo mật, tích hợp API/Webhook. | Triển khai hạ tầng riêng, domain white-label, bảo mật dữ liệu theo chuẩn doanh nghiệp. |
| **Extend Modules – Bám đuổi (Follow-up, Sequence)** | Automation | Có template sẵn; cần hiểu thời gian gửi & nội dung phù hợp. | Thiết kế flow bám đuổi đa bước, cá nhân hóa theo hành vi. | Tích hợp vào chiến dịch remarketing đa kênh, đồng bộ với quảng cáo. |
| **Extend Modules – GenAI, AI Chatbot, AI Seller** | Automation | Dùng mẫu AI có sẵn; không cần code. | Tùy biến prompt, huấn luyện dataset riêng, kết hợp AI vào kịch bản bot. | Xây dựng trợ lý ảo doanh nghiệp, tích hợp AI vào toàn bộ quy trình CSKH & bán hàng. |
| **Extend Modules – Meta Dataset, Meta CAPI, Meta Ads Manager, Meta Custom Audience** | Integration | Cần hướng dẫn kết nối; hơi kỹ thuật. | Hiểu cách đồng bộ dữ liệu quảng cáo, tracking chuẩn. | Tích hợp full-funnel marketing, đo lường & tối ưu ngân sách quảng cáo. |
| **Extend Modules – Payment Hub** | Integration | Dễ dùng nếu chỉ bật phương thức thanh toán có sẵn. | Tích hợp nhiều cổng thanh toán, xử lý logic đơn hàng. | Xây dựng hệ thống thanh toán đa kênh, đồng bộ ERP/kế toán. |
| **Extend Modules – Quản lý đơn hàng** | Tính năng chính | Dễ xem/trạng thái đơn; thao tác cơ bản. | Tích hợp với sàn TMĐT, kho vận, đồng bộ tồn kho. | Triển khai OMS (Order Management System) cho đa kênh bán hàng. |
| **Extend Modules – Lucky Wheel, Puzzle Game, Gift Opening, Quiz, Coupon** | Modules | Dễ tạo game mini từ template. | Tùy biến giao diện, logic quà, kết hợp automation. | Tích hợp gamification vào chiến dịch marketing lớn, thu thập data khách hàng. |
| **Extend Modules – Form Builder, Customize Image** | Modules | Kéo thả form, tạo ảnh đơn giản. | Tích hợp form vào bot, xử lý dữ liệu qua API. | Xây dựng hệ thống thu thập dữ liệu & cá nhân hóa nội dung marketing. |
| **Extend Modules – Messenger Live Shopping** | Modules | Dễ bật tính năng livestream bán hàng. | Kết hợp chatbot chốt đơn, automation follow-up. | Triển khai livestream commerce đa kênh, đồng bộ tồn kho & thanh toán. |
