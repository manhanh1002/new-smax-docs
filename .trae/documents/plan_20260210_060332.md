Tôi sẽ thực hiện kế hoạch sau để bổ sung tính năng "Share this docs" bên cạnh phần đánh giá:

1.  **Tạo component mới `DocsShare` (`components/docs/docs-share.tsx`):**
    *   **Giao diện:**
        *   Tiêu đề: "Share this docs"
        *   Các nút:
            *   **Copy URL**: Icon `Link` (hoặc `Copy`), click để copy link hiện tại vào clipboard.
            *   **Facebook**: Icon `Facebook`, click để mở cửa sổ chia sẻ Facebook.
            *   **Email**: Icon `Mail`, click để mở trình gửi mail mặc định (`mailto:`).
    *   **Logic:**
        *   Sử dụng `useCopyToClipboard` hook cho nút Copy URL.
        *   Sử dụng `window.open` với URL chia sẻ của Facebook: `https://www.facebook.com/sharer/sharer.php?u={url}`.
        *   Sử dụng thẻ `a` với `href="mailto:?subject={title}&body={url}"` cho nút Email.

2.  **Cập nhật `DocContent` (`components/docs/doc-content.tsx`):**
    *   **Import:** Import `DocsShare` component mới tạo.
    *   **Layout:**
        *   Thay thế thẻ `div` bao quanh `DocsRating` hiện tại bằng một `div` sử dụng `grid` layout (hoặc `flex`).
        *   Cấu trúc grid 2 cột trên desktop (`lg:grid-cols-2`), 1 cột trên mobile.
        *   Cột trái: `DocsRating` (giữ nguyên logic cũ).
        *   Cột phải: `DocsShare` (component mới).
        *   Thêm `gap` và `border` phù hợp để phân chia 2 khu vực nếu cần.

**Cấu trúc code dự kiến trong `doc-content.tsx`:**

```tsx
<div className="mt-8 grid gap-8 border-t pt-8 lg:grid-cols-2">
  <DocsRating slug={slug} />
  <DocsShare title={title} slug={slug} />
</div>
```

**Cấu trúc `DocsShare`:**

```tsx
export function DocsShare({ title, slug }: { title: string; slug?: string }) {
  // Logic lấy URL hiện tại
  // Render buttons
}
```

Kế hoạch này đảm bảo tính năng được thêm vào đúng vị trí, giao diện gọn gàng và đáp ứng đúng yêu cầu của bạn.
