Tôi sẽ thực hiện các thay đổi giao diện theo yêu cầu của bạn như sau:

1.  **Di dời thanh tìm kiếm lên Top Navigation Bar**:
    -   Sửa file `components/layout/topbar.tsx`: Thêm thanh tìm kiếm vào vị trí giữa Logo và Menu (hiển thị trên desktop). Trên mobile vẫn giữ icon tìm kiếm gọn gàng.
    -   Sửa file `components/layout/sidebar.tsx`: Xóa nút tìm kiếm khỏi Sidebar. Việc này sẽ tự động đẩy danh sách menu bên trái lên cao hơn.

2.  **Chuyển icon AI thành Widget**:
    -   Sửa file `components/layout/sidebar.tsx` & `components/layout/topbar.tsx`: Xóa các nút kích hoạt AI (icon Sparkles) hiện tại.
    -   Sửa file `components/layout/docs-shell.tsx`: Thêm một nút Widget nổi (Floating Action Button) ở góc dưới bên phải màn hình.
    -   **Hành vi**: Khi bấm vào Widget, nó sẽ mở `AssistantSheet` (Side Panel AI) và Widget sẽ ẩn đi (hoặc nằm dưới panel).

3.  **Loại bỏ thanh hỏi đáp AI trôi nổi**:
    -   Sửa file `components/layout/docs-shell.tsx`: Xóa component `AssistantDock` (thanh nhập liệu trôi nổi ở dưới cùng trang nội dung).

### Các file sẽ chỉnh sửa:
-   `components/layout/sidebar.tsx`: Xóa Search/AI buttons.
-   `components/layout/topbar.tsx`: Thêm Search bar, xóa AI icon.
