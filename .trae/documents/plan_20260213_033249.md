Tôi sẽ thực hiện thay đổi toàn diện giao diện người dùng (UI) dựa trên Design System mới, bao gồm cập nhật phông chữ, bảng màu và các thông số thiết kế trong cấu hình Tailwind CSS v4.

### 1. Cập nhật Phông chữ (Typography)
Thay thế font `Geist` hiện tại bằng các font được yêu cầu trong Design System:
- **Body/UI:** `DM Sans`
- **Headings:** `Sora`
- **Code:** `JetBrains Mono`

**Các bước thực hiện:**
- Sửa file `app/layout.tsx`:
  - Import `DM_Sans`, `Sora`, `JetBrains_Mono` từ `next/font/google`.
  - Định nghĩa các biến CSS (`variable`) cho từng font.
  - Áp dụng class biến vào thẻ `<body>`.

### 2. Cập nhật Design Tokens & Colors (Global CSS)
Chuyển đổi hệ thống màu từ OKLCH (hiện tại) sang bảng màu Hex của SmaxAi trong `app/globals.css`.

**Bảng màu mới (Light Mode):**
- **Primary (Brand):** `#f97316` (Cam)
- **Background:** `#f8f9fa` (Xám nhạt surface)
- **Foreground:** `#0f1835` (Navy đậm)
- **Sidebar:** `#ffffff` (Trắng)
- **Border:** `#e8ecf2`
- **Accent/Secondary:** `#fa6e5b` (Coral) hoặc `#f4f6fa` (Subtle) tùy ngữ cảnh.

**Bảng màu mới (Dark Mode - Dự kiến):**
- **Background:** `#0f1835` (Navy)
- **Foreground:** `#ffffff`
- **Sidebar:** `#0a1025` (Navy đậm hơn)

**Các bước thực hiện:**
- Sửa file `app/globals.css`:
  - Cập nhật khối `:root` và `.dark` với các giá trị màu mới.
  - Cập nhật khối `@theme inline`:
    - Gán `--font-sans` thành `var(--font-dm-sans)`.
    - Thêm `--font-heading` trỏ tới `var(--font-sora)`.
    - Gán `--font-mono` thành `var(--font-jetbrains-mono)`.
    - Cập nhật `--radius` và các biến màu khác.

### 3. Cập nhật Layout & Component Styles
Đảm bảo các thành phần giao diện sử dụng đúng font heading.
- Sửa `app/globals.css` (hoặc file cấu hình typography nếu có) để gán font `Sora` cho các thẻ tiêu đề `h1`, `h2`, `h3`, `h4`.

### 4. Kiểm tra
- Xác nhận không có lỗi build.
- Đảm bảo các biến CSS mới được áp dụng chính xác.
